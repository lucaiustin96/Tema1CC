1. Ce este PoliticieniDeTop.com?

PoliticieniDeTop.com este o platforma care are scopul de a aduna informatii despre politicieni intr-un singur loc si de a le face accesibile pentru utilizatori.

2. Maniera de interactiune cu utilizatorul

Utilizatorul poate:

  • Sa cauta stiri despre politicieni
  • Sa vada ce exprima fata politicianului cautat
  • Sa vada un top cu cei mai cautati politicieni
  • Sa caute un politician folosind vocea
1. Pe pagina principala utilizatorul are posibilitatea de a introduce numele politicianului despre care vrea sa afle mai multe informatii. De asemenea, utilizatorul poate sa vada care sunt cei mai cautati politicieni.
2. Pentru fiecare cautare se deschide o pagina noua. Pagina contine o poza care apartine persoanei cautate, expresiile faciale care se regasesc in poza si mai multe stiri care au legatura cu persoana cautata. Stirile sunt traduse in romana.

3. Detalii referitoare la implementare

Azure Database for MySQL server
Am folosit Azure Database for MySQL server pentru a stoca in tabelul "recent_searches" informatii despre cele mai recente cautari. Tabelul are patru coloane: id, Nume, Numar de cautari si Numar de reporturi. Pentru a ma conecta la baza de date trebuie sa folosesc un username si o parola care au fost generate atunci cand serviciul a fost ales pentru aplicatie.
Bing Image Search
Am folosit acest serviciu pentru a cauta o imagine care corespunde persoanei despre care utilizatorul vrea sa afle mai multe informatii. Serviciul cauta pe Bing mai multe imagini si returneaza un JSON, care este apoi parsat si din care este extrasa prima imagine.
Speaker Recognition API
Este folosit pentru a oferi utilizatorului posibilitatea de a cauta un politician, folosind comenzi vocale. Pentru a putea fi folosit, browser-ul va cere o permisiune. Serviciul primeste un input de tip audio si returneaza un json din care este extras textul care va fi folosit pentru cautare.
Face Recognition
Este un serviciu care primeste ca input o imagine si returneaza un json cu informatii. Aplicatia foloseste acel json pentru a extrage varsta si expresiile faciale care corespund pozei date ca input.
Bing News Search
Am folosit acest serviciu pentru a afisa o lista cu stiri despre persoana cautata. Serviciul primeste ca input un nume dupa care se face cautarea si returneaza un json care contine stiri despre persoana cautata. Json-ul contine o descriere pentru fiecare stire, un url si un titlu. Titlul si descriere sunt returnate in engleza.
Text Translate
Serviciul este folosit pentru a traduce un text. Aplicatia il foloseste pentru a traduce titlul si descrierea stirilor din engleza in romana. Ca si input este dat textul in engleza, iar ca outpute este returnat un json care contine textul tradus in romana.
